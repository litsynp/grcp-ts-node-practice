# gRPC - TypeScript - Node Practice

This is a practice project for gRPC with Node.js and TypeScript.

## How to run

### Generate proto files

```bash
$ chmod +x ./proto-gen.sh
$ npm run proto:gen
```

### Run the gRPC server

```bash
$ npm run server
```

### Run the gRPC client

```bash
$ npm run client
```

## How to run (with build)

### Build

```bash
$ npm run build
```

### Run the gRPC server

```bash
$ node dist/server
```

### Run the gRPC client

```bash
$ node dist/client
```
